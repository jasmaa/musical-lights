Instructions: Click "run" to start with a default song (ideally 90-120 bpm). You can upload files while it's running to overwrite the currently
playing song, but this will pause it: you will have to click the play button again. Alternatively, click the checkbox before
clicking "run" in order to use the mic. If you've already run the program on audio and want to restart, refresh the page.
